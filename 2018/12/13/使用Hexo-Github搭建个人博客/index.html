遇到的几个问题：

1、配置后网页无法显示主题

在更改网站的主题后，生成网页并本地成功预览，然而当完成部署后（并没有报错）,博客网站加载主题格式失败，网页布局混乱。

解决方法：
在站点配置文件（根目录下的_config.yml）中，更改url和root两个属性的值，如下所示：

url: http://fanyangfanyang.github.io/blog
root: /fanyangfanyang.github.io/
